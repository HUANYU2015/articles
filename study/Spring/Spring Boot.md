# Spring boot

$特点$

* Spring boot 可以创建独立运行的应用而不依赖与容器
* 不需要打包成 war 包，可以直接放在 Tomcat 中直接运行
* 简化配置，不用再看过多的 xml
* 为 Spring Cloud 铺路，SpringBoot 可以整合很多各式各样的框架来构建微服务，比如 dubbo,thrift 等

$使用场景$

* 有 Spring 的地方
* J2EE/Web 项目
* 微服务
